# Getting Started with Digital Kòrsou

**Bon bini! Welcome to the Boske di AI**

---

## What is Digital Kòrsou?

Digital Kòrsou is an AI ecosystem designed for Curaçao. It consists of multiple specialized AI agents (GPTs) that help with education, employment, health, culture, and more – all in Papiamentu.

---

## Quick Start

### For Users

Simply interact with the AI in Papiamentu (or Dutch/English/Spanish):

```
"Bon dia! Mi ta buska yudansa ku mi tarea di matemátika."
```

The system will route you to the right agent automatically.

### For Developers

1. Clone the repository:
```bash
git clone https://github.com/sahidattaf/digital-korsou-boske-ai.git
cd digital-korsou-boske-ai
```

2. Install dependencies:
```bash
cd api && npm install
```

3. Set up environment:
```bash
cp .env.example .env
# Add your API keys
```

4. Run locally:
```bash
npm run dev
```
