# Contributing to Digital Kòrsou – Boske di AI

**Danki pa bo interes!** Thank you for your interest in contributing to Digital Kòrsou.

---

## Before You Begin

This project serves the people of Curaçao. Before contributing, please understand:

1. **Papiamentu is sacred** – This is the mother tongue of Curaçao. Any language contributions must be reviewed by native speakers.
2. **Cultural sensitivity matters** – Symbols, names, and metaphors carry meaning. Research before suggesting changes.
3. **Community over code** – We prioritize human benefit over technical elegance.

---

## How to Contribute

### 1. Report Issues
- Use GitHub Issues for bugs, suggestions, or questions
- Include context: what you expected vs. what happened
- Screenshots or logs help

### 2. Suggest Features
- Open a Discussion or Issue with your idea
- Explain how it benefits the Curaçao community
- Be open to feedback

### 3. Submit Code
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly
5. Submit a Pull Request

### 4. Improve Documentation
- Fix typos, clarify instructions
- Add examples
- Translate to Papiamentu, Dutch, or Spanish

### 5. Add Knowledge
- Educational content for agents
- Local information (verified)
- Papiamentu vocabulary and phrases

---

## Code Standards

- **Language:** TypeScript for API, Markdown for documentation
- **Formatting:** Use Prettier defaults
- **Comments:** In English, clear and concise
- **Commits:** Descriptive messages in English

---

## Papiamentu Guidelines

If contributing Papiamentu content:

- Use standard orthography
- Avoid mixing with Dutch/English unless natural
- Verify with native speakers when possible
- Include translations for non-speakers

---

## Review Process

1. All PRs require at least one review
2. Language contributions require native speaker review
3. Be patient – we're a small team
4. Be respectful in discussions

---

## Code of Conduct

- Be respectful
- Be inclusive
- Be constructive
- No harassment, discrimination, or harmful content
